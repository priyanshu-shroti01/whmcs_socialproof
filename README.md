# 📈 Conversion Benefits

Using social proof on your website can significantly improve visitor trust and increase sales.

### Why Social Proof Works

When visitors see real-time activity such as recent purchases or signups, it creates **trust, urgency, and credibility**.

This psychological effect encourages visitors to take action faster.

---

### Key Benefits

- **Increase Conversion Rates**  
  Real-time purchase notifications encourage visitors to complete their orders.

- **Build Customer Trust**  
  Showing recent activity reassures visitors that other customers are actively buying your services.

- **Create FOMO (Fear of Missing Out)**  
  Visitors feel urgency when they see others purchasing products.

- **Improve Engagement**  
  Dynamic notifications keep users interested and interacting with your website.

- **Boost Product Visibility**  
  Highlight popular products automatically through simulated or real activity.

# 📦 Installation

1. Upload files to

```
/modules/addons/
```

2. Login to **WHMCS Admin Panel**

```
System Settings → Addon Modules
```

4. Activate **ShrotiHost Social Proof**

5. Configure module settings.

---

# 🚀 Usage

After activation, the module will start displaying **recent activity popups** on your WHMCS frontend.

You can configure:

- Popup appearance
- Display timing
- Simulated entries
- Sound notifications (Premium)
- Product priority generator (Premium)
